# 🎯 Number Guessing Game

## 📌 Project Description
This is a Python-based number guessing game where the computer generates a random number between 1 and 100.  
The user has 4 chances to guess the correct number.

## 🚀 Features
- Random number generation (1 to 100)
- 4 attempts to guess the number
- Hint system (Too High / Too Low)
- Displays success message when guessed correctly
- Shows correct answer if user fails

## 🛠️ Technologies Used
- Python

## ▶️ How to Run
1. Install Python
2. Save the file as `number_guessing_game.py`
3. Run using:
   python number_guessing_game.py
